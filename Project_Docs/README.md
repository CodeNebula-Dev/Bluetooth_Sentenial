# Comprehensive Project Documentation Bundle

Welcome to the comprehensive documentation folder for the **Ghost Lock - Stealth BLE Proximity Lock System**.

This folder contains detailed explanations of the project phases and in-depth breakdowns of every single code file in the project.

## Folder Structure

- **`Project_Phases.md`**: Outlines the simulation and hardware phases of the Ghost Lock project.
- **`Code_Explanations/`**: Contains a markdown file for every important source file, explaining its purpose, logic, and how it fits into the overall system.

## Project Overview

Ghost Lock is an invisible Bluetooth Low Energy (BLE) proximity lock system that uses your phone as a secure, silent key without requiring an app or button presses.
